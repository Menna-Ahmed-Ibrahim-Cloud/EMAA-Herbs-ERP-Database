# 🌿 EMAA Herbs ERP System - Database Part
This project is a comprehensive ERP system designed for managing organic herbs, developed as a graduation project for Thebes Higher Institute.

## 🛠️ My Technical Role
I was responsible for the **Database Engineering** phase, ensuring a robust and scalable data architecture.

### Key Responsibilities:
* **Database Design:** Created the full schema using **Microsoft SQL Server**.
* **Data Modeling:** Designed the **Entity-Relationship Diagram (ERD)** to define complex relationships between products, suppliers, and orders.
* **Integrity & Security:** Implemented data constraints and roles to ensure data consistency across the ERP modules.
* **Methodology:** Followed the **Waterfall Methodology** for systematic database development.

## 📂 Project Structure
* `/SQL-Scripts`: Database creation and query files.
* `/Documentation`: Full project report and ERD diagrams.
